# -Currency-Converter-
💱 Currency Converter  A simple and responsive Currency Converter Web App built using HTML, CSS, and JavaScript. It allows users to convert currencies in real-time with country flags support.

## 🌐 Live Demo
https://suspect-911.github.io/-Currency-Converter-/
---

## 🚀 Features

- 🌍 Convert between multiple currencies  
- 🇺🇸🇮🇳 Dynamic country flags based on selection  
- 🔄 Real-time exchange rates (API-based)  
- 📱 Responsive design (works on mobile & desktop)  
- ⚡ Fast and lightweight  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Currency API (ExchangeRate API or similar)  
- Flag API

## 🔧 How It Works

- User selects currencies  
- App fetches exchange rates from API  
- JavaScript calculates and displays result  
- Flags update dynamically using country codes
